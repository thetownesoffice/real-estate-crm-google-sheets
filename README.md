# Real Estate CRM — Google Sheets

**Industry:** Real Estate  
**Timeline:** 4 days  
**Result:** 8 active buyers converted from a pipeline that had previously produced zero

---

## The Problem

A solo real estate agent was managing her entire buyer pipeline on a 
handwritten sign-in sheet. There was no follow-up system, no way to 
track where each buyer was in the process, and no visibility into her 
pipeline at all.

---

## What I Built

A fully functional AI-assisted CRM inside Google Sheets, built from 
the ground up. It includes:

- Color-coded status tracking across all pipeline stages
- Automated follow-up alerts so no buyer goes cold
- A live pipeline dashboard with full visibility at a glance

---

## Tools Used

- Google Sheets
- AI-assisted build process
- Conditional formatting and logic for status tracking
- Dashboard design for pipeline visibility

---

## The Result

Eight active buyers converted from a list that had previously 
produced zero results.
